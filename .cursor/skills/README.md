# N.E.O. Cursor agent skills

Each **skill** is a directory named **`skill-name`** containing **`SKILL.md`** with YAML frontmatter (`name`, `description`). The `name` field matches the folder name.

Cursor picks these up from **`.cursor/skills/`** automatically. If a skill is missing in **Settings → Rules → Agent Decides**, reload the window or restart Cursor.

| Folder | Chat shortcut |
|--------|----------------|
| `neo-supabase-migration-pass` | `/neo-supabase-migration-pass` |
| `neo-verify-gates` | `/neo-verify-gates` |
| `neo-ui-preservation` | `/neo-ui-preservation` |
| `neo-server-hardening` | `/neo-server-hardening` |

Authoritative summaries and “when to use” guidance: root **`AGENTS.md`**.
